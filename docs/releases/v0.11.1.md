# v0.11.1 - Vercel Create Room Hotfix

## Summary

This patch release fixes the online room creation flow on deployed environments such as Vercel.

## Fixed

- Hardened the `/online/create` page so it no longer assumes every failed response contains valid JSON.
- Added safer parsing for empty, text, or malformed API responses.
- Updated the create-room API to return a structured JSON error when Supabase server environment variables are missing.
- Improved user-facing Chinese error messages for create-room failures.

## Notes

- No game rules were changed.
- No local game behavior was changed.
- Online multiplayer remains an MVP and still requires Supabase environment variables and schema setup.
